Programação de Soluções Computacionais

Essa unidade é fundamental para a formação de um desenvolvedor. Nela, os alunos aprendem as bases da programação, utilizando linguagens como Java, Python ou C#. São abordados conceitos como algoritmos, estruturas de dados, lógica de programação e desenvolvimento de software

Modelagem de Software

A modelagem de software é uma etapa crucial no desenvolvimento de sistemas. Nesta unidade, os alunos aprendem a utilizar diagramas e notações para representar a estrutura e o comportamento de um sistema de software. UML (Unified Modeling Language) é uma ferramenta comum utilizada nesta área.

Usabilidade, Desenvolvimento Web, Mobile e Jogos

Nesta disciplina, o foco está no desenvolvimento de interfaces de usuário intuitivas e agradáveis. Os alunos aprendem sobre design de interfaces, desenvolvimento web, desenvolvimento de aplicativos móveis e desenvolvimento de jogos.

Ambientes Computacionais e Conectividade

Esta unidade aborda os fundamentos de redes de computadores, protocolos de comunicação e infraestrutura de TI. Os alunos aprendem sobre como os computadores se conectam e como os dados são transmitidos entre eles.

Digital Personalizável

Esta unidade aborda o desenvolvimento de sistemas personalizados e adaptáveis às necessidades dos usuários. Os alunos aprendem sobre técnicas de inteligência artificial, aprendizado de máquina e análise de dados para criar sistemas mais inteligentes e personalizados.

Sistemas Computacionais e Segurança

Esta unidade curricular provavelmente aborda os fundamentos da arquitetura de computadores, sistemas operacionais, redes de computadores e segurança da informação. Os alunos aprendem sobre os componentes físicos e lógicos de um computador, como eles interagem e como proteger os dados e sistemas contra ameaças.

Sistemas Distribuídos e Mobile

Nesta disciplina, o foco está em sistemas de software que se distribuem por múltiplos dispositivos ou computadores, como redes e aplicações móveis. Os alunos aprendem sobre protocolos de comunicação, desenvolvimento de aplicativos para dispositivos móveis e desafios relacionados à segurança e escalabilidade em sistemas distribuídos.
